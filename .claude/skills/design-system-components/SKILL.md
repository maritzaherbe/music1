# Design System Components

Full spec: `research/skills.md` #19.

## Procedure
1. Build primitives in `src/components/` with NativeWind (Tailwind) tokens.
2. Support light/dark + WCAG 2.1 AA (contrast, focus, labels).
3. Include loading/empty/error states used across screens.
4. For any charts/marketing visuals, prefer the existing `dataviz` / `artifact-design` skills.
